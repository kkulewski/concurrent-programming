## Cient - Server communication using text files + lockfile
Compile:

`gcc -o client client.c`

`gcc -o server server.c`


Run:

`./server {SERVER_NAME}`

`./client {SERVER_NAME}`

Client - input (Ctrl+D, then Enter to end input):

`{CLIENT_NAME}`

`{MESSAGE}`

Server - input (Ctrl+D, then Enter to end input):

`{RESPONSE}`
