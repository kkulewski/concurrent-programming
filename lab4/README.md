## Client - Server communication using FIFO (bash)

Run server:

`./server`


Run client:

`./client SERVER_USER_NAME ARGUMENT`

### Related

Show all running processes (a - all users, u - owner, x - not attached):

`ps aux`


Send signals:

`kill -SIGTERM PID`

`kill -SIGCHLD PID`

`kill -SIGUSR1 PID`
